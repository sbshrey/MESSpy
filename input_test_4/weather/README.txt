WEATHER DATA - INPUT_TEST_4

This directory contains weather data files for the Mixed End-Use Hybrid Plant simulation.

Files:
- TMY_general.csv: Typical Meteorological Year data for general location
- TMY_general15.csv: 15-minute resolution TMY data for detailed analysis

The weather data is used for:
- Wind power generation calculations
- Solar PV performance modeling
- System efficiency analysis under varying weather conditions

Location: Latitude 42.33°, Longitude 10.489° (Mediterranean region)
Time Zone: UTC+1 with Daylight Saving Time
Data Resolution: 1-hour and 15-minute intervals
